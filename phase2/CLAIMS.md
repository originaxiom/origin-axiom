# Phase 2 Claims Register (Locked)

This document enumerates **all scientific claims made in Phase 2** and maps each claim to
**canonical, reproducible artifacts**.

**Rule:** No claim is valid unless it is backed by at least one canonical artifact listed
below.

Phase 2 claims are restricted to **existence, robustness (over tested ranges), and
consistency/viability**. Phase 2 does **not** claim empirical confirmation, uniqueness, or
a derived value of the observed cosmological constant.

## Claim 2.1 (C2.1) — Existence of a vacuum-energy proxy residual

**Statement (allowed scope)**

In the Phase 2 finite mode-sum toy model, the computed vacuum-energy proxy residual is
**non-zero** under the declared model and configuration.

The model includes an Origin-Axiom-inspired global non-cancellation floor parameterized by
ε, but the existence claim does **not** require that the ε-floor is active; it asserts
only that the reported residual output is non-zero for the canonical configuration.

This is an **existence** claim: it asserts non-zero residual output under the declared
model and configuration, not a derivation of the observed cosmological constant.

**Status**: Demonstrated (numeric)

**Primary artifact**

  - Metadata: `meta.json`, `params.json`, `summary.json`

## Claim 2.3 (C2.3) — Cosmological viability in standard flat FRW (explicit mapping)

**Statement (allowed scope)**

Interpreting the Phase 2 residual as an **effective Λ-like component** in standard flat
FRW dynamics yields **Λ-like late-time acceleration behavior** for the configured
cosmological parameters.

This is a **consistency/viability** claim: it shows that the residual can be inserted into
standard FRW equations without modifying GR and can yield Λ-like behavior, but it does not
claim a fit to observational data.

**Status**: Demonstrated (numeric FRW wrapper)

**Primary artifact**

- **Figure E** — `outputs/figures/figE_frw_comparison.pdf`

**Provenance pointer**

- Run pointer: `outputs/figures/figE_frw_comparison.run_id.txt`
- Canonical run_id: `figE_frw_comparison_20251227T223712Z`
- Run data: `outputs/runs/<run_id>/raw/frw_timeseries.npz`
- Produced figure: `outputs/runs/<run_id>/figures/frw_comparison.pdf`

**Operational mapping (binding)**

- Mapping factor: `residual_to_omega_lambda = 0.01`
- Effective cosmological constant proxy:
  `Omega_L_eff = residual_to_omega_lambda × residual_constrained`

Closure policy (binding):
- `cap_mode = closure` enforces flatness via:
  `Omega_m_eff = 1 − Omega_r − Omega_L_eff`

**Observed numeric outcome (binding)**

From `outputs/runs/<run_id>/summary.json`:

- `residual_constrained = 1.8640648476264552`
- `residual_to_omega_lambda = 0.01`
- therefore `Omega_L_eff = 0.01864064847626455`
- `Omega_r = 0.0`
- closure gives `Omega_m_eff = 0.9813593515237354`

Figure E compares:
- baseline flat FRW (`Omega_L_base = 0.0`, `Omega_m_base = 1.0`)
- OA_eff (the mapped model above)
- a reference curve (`Omega_L_obs = 0.7`, `Omega_m_obs = 0.3`), for visual context only

## Explicit Non-Claims (binding)

Phase 2 explicitly does NOT claim:

- a derivation of the observed value of the cosmological constant,
- an exact numerical match to observation,
- a microscopic origin of the interference phase,
- a definitive resolution of the cosmological constant problem,
- modified gravity or new gravitational dynamics,
- Standard Model / flavor / neutrino connections,
- early-universe dynamics or structure formation,
- unification narratives.
