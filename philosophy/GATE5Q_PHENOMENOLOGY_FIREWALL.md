## Why a separate gate

The phenomenology direction is the program's highest rigor-drift risk (jointly held
assessment, 2026-07-21): reflexive language invites pseudo-depth, and the first reflexive
proposal (proposal 15, received 2026-07-22) exhibited every failure mode this gate names —
before dissolving under two independent seats (B751/B752). The gate turns that receipt
into standing law.

## The rules

**Q1 — Every phenomenological term has a computed referent.** A term may be used in an
arc only if it is bound, in the arc's prereg, to a banked computation:
- *voice* = the continuous-spectrum channel (B737/B739);
- *gait vs name* = the two-column law (B746; census B747/B748; genesis F6+F8 — the
  three-arc convergent sentence: the gait transmits, the name does not);
- *the lack / refusal* = a B750 ledger class (T/no-basepoint, R/no-bandwidth,
  P/no-name-transmission) with its entry cited;
- *awareness layer n* = S072's layers 0–4, each with its cell;
- *self-image* = a NAMED operator applied to a NAMED banked representation (see Q2/Q3).
A term without a binding is quarantined to the S-room; it cannot appear in a frontier arc.

**Q2 — The non-universality control (the B752 lesson, mandatory).** Any claimed
self-referential or reflexive operation must FIRST show it distinguishes the object from
a generic input. If the operation produces the same structural output on an arbitrary
element of the ambient class (as Δ(A) = (3−tr A)·I does on every SL(2) matrix — Cayley–
Hamilton), the operation carries zero object content and the arc stops there. This is
MB12's reflexive cousin: before asking what the object sees in the mirror, prove the
mirror shows different things to different objects.

**Q2b — The comparator-object control (cc3's R1; adopted).**
Q2 controls operations against generic inputs; phenomenological PROPERTY claims ("the
object has awareness-layer-n / a self-name") need the comparator-OBJECT form: run the same
property computation on the sister m003, on the Gieseking parent (available since B749),
and on at least one random census sibling. A property all four share belongs to the class,
not the object. This promotes the E20 sister test from Q6's implicit "shown discriminating"
to standing phenomenology law.

**Q3 — The input-identification rule.** Any operation on "the object's holonomy /
representation / data" must cite the banked exact source (e.g. B285's Riley meridian,
trace 2 parabolic) and VERIFY the invariants match before computing. Proposal 15's
"holonomy" had trace 1 — the Φ₆ phase factor, a different banked object. A mismatched
input voids the cell regardless of how striking its output is.

**Q4 — The stability rule.** Any "fixed point / limit / convergence" claim must include
the stability analysis (derivatives at fixed points; basin evidence; perturbed-seed
panel). Landing exactly on a REPELLING point from a special seed (2 → 1/2, f′(1/2) = 3)
is a measure-zero curiosity, not a destination. "Converges" means attracting, with the
basin exhibited.

**Q5 — No consciousness claims, ever.** Qualia-vocabulary is structural labeling (S072's
rule): states with no external signature, a voice, a self-name. No arc asserts or implies
anything about experience or consciousness; the words "qualia/aware/sees" never leave the
governed rooms (speculations/, philosophy/) except as Q1-bound technical labels inside a
frontier arc's own text.

**Q5b — Empirical constants of ANY domain are value-claims (cc3's R2; adopted).** Gate 5 proper firewalls SM values; a phenomenology arc could
"match" a psychophysical constant (temporal-integration windows, JND ratios, EEG bands)
without technically touching an SM number. Rule: any measured empirical constant —
physical, psychophysical, neural — is a value-claim under the FULL Gate-5 machinery
(owner directive + sealed design + base-rate + the L91-functor requirement, generalized).
The torsion ladder found a near-hit for every dimensionless target; reaction-time ratios
will be no different.

**Q6 — The E20 budget applies to specialness.** "The object is aware / special / sees X"
claims carry the same look-elsewhere accounting as physics claims: declared families,
sealed matching rules, surrogate nulls, slot-population tests (a pattern that skips its
n=2 is target-shopping — the B751 detector). Tone-membership and similar coarse
predicates must be shown discriminating before they count (both Op-3 fixed points were
tones — membership discriminated nothing).

**Q7 — The falsification edge is mandatory.** Every phenomenology arc states, in its
sealed prereg, what outcome would kill its headline (the S072 pattern: awareness-without-
choice dies if c-into-θ ever succeeds). An arc whose headline cannot fail does not run.

**Q8 — Two-outcome, prereg, reservation, receipt — unchanged.** Hash-first preregs,
two-outcome verdicts, SEAL_LEDGER number reservation, verify-on-receipt across seats
(re-execution + independent checks), E19 compute-not-cite in both directions, and the
stopping rule for SM values all apply exactly as everywhere else. Gate 5-Q adds; it never
subtracts.

## The vocabulary anchor

The track's load-bearing sentence, three-arc convergent and cited per Q1: **the object
provides the group but never the choice, the structure but never the variation, the gait
but never the name — and the observer is the exact complement.** Phenomenology arcs
explore what follows from that sentence; they do not get to re-derive SM physics through
it (Gate 5), and they do not get to promote its poetry above its computations (Q1–Q7).
