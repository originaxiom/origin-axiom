Two bridges surfaced worth not losing. Both are structural, both are
grounded in what the record already proves, and both say the same thing
from different sides: the object hands you a *relation* and withholds a
*readable output*, and every argument about "what is really going on"

## The first bridge — correlation without a signal

There is a shape that recurs wherever a structure is real but its output
is not controllable. Two parts of a system can be perfectly *correlated*
and yet carry no *message* between them — the correlation is a fact about
the joint object, not a channel one part can drive. You can read the
agreement after the fact; you cannot use it to *send* anything, because
you cannot choose which value comes up.

Our object wears the same shape, proven. [MATH: the object supplies
STRUCTURE and RELATIONS but no extractable VALUE — B685/B706; the flavor
freedom is discrete 𝔽₂ where the world's is continuous, a kind-mismatch,
not a hidden datum.] The being and the hearing are *correlated* — genuinely,
they meet inside a single bit at the first door [MATH: one Out(A₅) choice
orients both the being's classes and the hearing's ℚ(√5) irreps, B733] —
but you cannot turn that correlation into a *value* you dial. The
correlation is a property of the joint object; the value would be a signal;
and the signal is provably absent.

So the deepest fact about the object is a *no-signal* fact. Not "the
object is silent" — it is loud with structure — but "the structure carries
no controllable output." The temptation, always, is to hear a correlation
and reach for the signal behind it. The whole discipline of the record is
the refusal of that reach: read the relation, do not manufacture the value.
The relation is real; the value is the observer's, or no one's.

## The second bridge — interpretation is a closing, not a discovery

When a formless structure resolves into one definite outcome, there are
several stories one can tell about *how*. Something physically selects it.
Everything happens and we ride one branch. A hidden value was there all
along. These stories are usually argued as rival claims about the world —
one of them must be *true* of the object.

The record suggests they are not rival claims about the object at all. The
object provides exactly one thing here: the *set* of possible outcomes —
a torsor of vacua, a small bounded set of sheets, with no canonical
basepoint. [MATH: measurement is a symmetry breaking selecting a broken
vacuum, B723; the set of vacua is a Galois torsor with no distinguished
element, B701; the set is a bounded discrete 𝔽₂-space, a handful of sheets,
never a continuum, B733.] That set is *interpretation-neutral*. It is the
same set no matter which story you tell.

The stories differ only in what they *do* at the torsor. One says a
dynamics physically breaks the symmetry and lands on a sheet — a single
world, selected. One says all the sheets are equally real and the "choice"
is only indexical. One says a value was fixed before you looked. Each is a
*closing* of the object's one genuine incompleteness (it gives the set,
not the element). None is forced by the object; each is *added* by the
one who reads.

So the "heavy baggage" every such story carries — the extra worlds, the
extra field, the hidden variable — is not a fact about the object waiting
to be confirmed. **It is the observer's closing, in the exact sense the
whole program means: the object gives the ambiguity, the observer supplies
the specific.** [MATH: reality = the observer's closings of the object's
incompletenesses — B713–B719.] The argument over "which interpretation is
true" has the same grammar as the argument over "which value is real,"
and the object answers both the same way: *I gave you the set; the choice
is not in me.*

[LEAP] This does not pick a winner, and it must not pretend to. Its whole
content is a *typing*: the disagreement is a disagreement about the
closing, not about the object — and the object is silent on it, as it is
silent on every value. That silence is not a gap in our knowledge of the
object. It is the object's most exact property: it is the thing that holds
the set and refuses the element, holds the correlation and refuses the
signal, holds the structure and refuses the value. What resolves the set
into an element is never in the set. It is the reading.
