\section{Fit Pipeline: CKM+PMNS \texorpdfstring{$\rightarrow$}{->} $\theta$}
We define an explicit ansatz mapping $\theta$ to target observables and perform a grid-based $\chi^2$ scan.
We report best-fit $\theta$, an uncertainty interval, and diagnostics.

\subsection{Discrete offset sweep and baseline choice}
To avoid hidden parameter fitting, the Phase 3 baseline fixes all offset choices a priori and fits only $\theta$.
We nevertheless test a small discrete set of candidate fixed offsets for the PMNS phase mapping,
$b_{\mathrm{PMNS}} \in \{0,\ \pi,\ -\pi/2,\ +\pi/2\}$, holding the rest of the ansatz fixed.
For each hypothesis we run the same grid scan over $\theta$ and compare the minimum $\chi^2$ values.
The resulting model-selection evidence is reported in Appendix~\ref{tab:offset-sweep}.
We adopt as the Phase 3 baseline the discrete hypothesis with the lowest $\chi^2$ (under the frozen CKM/PMNS targets),
which selects $b_{\mathrm{PMNS}}=\pi$ for the current exploratory add-on.
