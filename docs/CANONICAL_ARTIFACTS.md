## Canonical vs Exploratory

### Canonical
- Listed explicitly in this document
- Generated by the Gate 0 reproducibility entrypoint
- Must include full provenance metadata
- Must not be silently overwritten

### Exploratory
- Parameter sweeps
- Debug plots
- Intermediate `.npz` scans
- Ad-hoc notebooks
- Scratch figures

Exploratory artifacts may be deleted or regenerated without notice.

---

## Overwrite Policy

Canonical artifacts:
- **must not** be overwritten silently
- require explicit user intent (`--overwrite`) or version bump
- must change `git_head` and `timestamp_utc` if regenerated

---
