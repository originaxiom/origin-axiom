## The derivation chain (each step is verifiable)

### Step 1: The Möbius map (exact, from P1)

A = [[2,1],[1,1]] acts on the upper half-plane H by:
  τ → f(τ) = (2τ+1)/(τ+1)

This is a standard Möbius transformation. The fixed points satisfy:
  (2τ+1)/(τ+1) = τ  →  τ² - τ - 1 = 0  →  τ = φ and τ = -1/φ

**Verification:** `f(phi) = (2*phi+1)/(phi+1) = phi` (exact).

### Step 2: The generating vector field (exact, from log(A))

The continuous flow generated by A on H is:
  dτ/dt = v(τ) = b + (a-d)τ - cτ²

where a, b, c, d are entries of log(A) = (log(φ²)/√5) × [[1,2],[2,-1]].

Since a = d (wait, a=scale, d=-scale, so a-d = 2*scale) and b = c = 2*scale:
  v(τ) = 2κ + 2κτ - 2κτ² = -2κ(τ² - τ - 1) 

where κ_half = log(φ²)/√5. Setting κ = 2·log(φ²)/√5:
  v(τ) = -κ(τ² - τ - 1)

**Verification:** `v(phi) = -κ(phi²-phi-1) = -κ·0 = 0` (fixed point).
`v(-1/phi) = -κ(1/phi²+1/phi-1) = -κ·0 = 0` (fixed point).
`v(0) = -κ(0-0-1) = κ > 0` (pushed toward φ).

This is the vector field on H generating the one-parameter family
exp(t·log(A)), which at t=1 gives back A. Exact.

### Step 3: The potential (exact, by integration)

Since v(τ) = -dV/dτ (gradient flow), the potential is:
  V(τ) = κ(τ³/3 - τ²/2 - τ) + const

Properties (all verifiable by differentiation):
  V'(τ) = κ(τ² - τ - 1)
  V'(φ) = 0  (minimum, since V''(φ) = κ(2φ-1) = κ√5 > 0)
  V'(-1/φ) = 0  (maximum, since V''(-1/φ) = -κ√5 < 0)
  V'(0) = -κ ≠ 0  (zero is NOT a critical point)

### Step 4: The field equation (standard promotion to field theory)

Promoting τ to a field τ(x,t) and adding a kinetic term gives:
  L = (1/2)(∂_μ τ)(∂^μ τ) - V(τ)
