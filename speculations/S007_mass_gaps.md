# S007 — mass gaps = the gap-labeling structure of the SL(n) operator

**Status: `POSTULATED`.** Firewalled; not a claim.

**Structural fact (cited, external).** The framework *is* a theory of gap structure: the Fibonacci spectrum is a
zero-measure Cantor set whose gaps are organized by the **gap-labeling theorem** (the K-theory of the substitution;
Bellissard — an external, known result, used in the repo only as a sanity-check control, not a new theorem).

**[LEAP] (kept separate).** Read a Yang–Mills-style "mass gap" as a **spectral gap**, for which we have an exact
labeling technology; and read raising the rank `n` as opening/closing gaps in the multichannel operator.

**[HOOK] (bounded calc, uncomputed).** Compute the gap structure of the **non-principal SL(n)** operator (the
multichannel / off-principal sector, `S011`): does raising the rank open or close gaps in a way the gap-labeling
group predicts? This is computable and is where genuinely new spectral content, if any, would appear.

**Negative beside the leap.** The gap-labeling is external known math; nothing here is a Yang–Mills result. The
"mass" reading is a Tier-1 *structural* analogy (gaps exist and are labelable), not a particle spectrum — and it
must avoid the killed identification `tower-eigenvalues = masses` (`S015`): this concerns the *operator's* spectral
gaps, not the tower's monodromy eigenvalues.
