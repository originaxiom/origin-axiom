# 04 — The unification (`ρ_n`)

> Narrative, not claim. Cites the mathematics one-way.

The "are they one object?" question had a clean answer: **yes — and the object is `ρ_n`.** Everything the program
had built — the tower (chapter 02), degree=rank and the A-polynomials (chapter 03), the parity sectors — are
sub-structures of a single `GL(2,ℤ)`-representation `ρ_n` on the SL(n) trace ring (**B103**, V87). The
factor-through-the-abelianization argument made `char(ρ_n)` a class function = the catalog *for all `n`*, so
"prove the all-`n` tower" became "**decompose `ρ_n`**."

A complementary route did the decomposition by hand for low rank. The Dehn-twist generators `U, L, S` of the
mapping-class group, composed in the eps-series, computed `char(ρ_n)` *without* the heavy Procesi trace ring, and
proved the explicit SL(4) catalog for *all* monodromies — metallic and non-metallic (**B104**, V88). The
opposition involution `θ = −w₀` supplied the multiplicities structurally: the height-2 root spaces split
`(2,0), (2,2), (4,2)` at `n = 3, 4, 5` (**B62**, V13), giving `char(M²)`, `char(M²)·char(−M²)`,
`char(M²)²·char(−M²)`.

So the sprawl collapsed to one target, sharpened over several audits (V90/V91): **prove `char(ρ_n) = the Dickson
catalog` by reproducing the opposition-involution multiplicities directly from `ρ_n`** — engine-free, around the
`σ`-construction rather than through it. That target is, to this day, the central open math problem of the project
(chapter 08).
