# PC12 Certificate Appendix

Status: certificate summary. This file records what the B48 probe checks. It is
not a substitute for a fully human-readable proof.

## Object

For `m >= 1`, define the free-group automorphism

```text
phi_m(a)=a^m b
phi_m(b)=a
```

and act on the standard rank-two `SL(3,C)` trace generators:

```text
x1 = tr(A)          x2 = tr(B)          x3 = tr(AB)
x4 = tr(A^-1)       x5 = tr(B^-1)       x6 = tr(A^-1 B)
x7 = tr(A B^-1)     x8 = tr(A^-1 B^-1)
```

## Trace-Map Recurrences

The trace map is generated by Cayley-Hamilton recurrences:

```text
tau_k   = x1 tau_{k-1} - x4 tau_{k-2} + tau_{k-3}
sigma_k = x4 sigma_{k-1} - x1 sigma_{k-2} + sigma_{k-3}
```

with

```text
tau_-1=x6, tau_0=x2, tau_1=x3
sigma_-1=x7, sigma_0=x5, sigma_1=x8
```

The induced map is:

```text
T_m = (tau_m, x1, tau_{m+1}, sigma_m, x4, sigma_{m-1}, tau_{m-1}, sigma_{m+1})
```

For `m=1`, this specializes to:

```text
(x3, x1, x1*x3 - x4*x2 + x6, x8, x4, x5, x2, x4*x8 - x1*x5 + x7)
```

## Certificate Checks

B48 verifies seven independent checks:

```text
1. m=1 formula and recurrence sanity
2. direct exact SL(3,Z) matrix trace sanity for m<=4
3. entropy degree recurrence and dominance checks for m<=25, n<=30
4. c=3 Jacobian sector block characteristic polynomials for m<=50
5. c=1 antisymmetric block characteristic polynomial by residue blocks r<=20
6. integer fixed-line splitting classification over an audit rectangle
7. compact SU(3) diagonal-slice representatives for c=-1,0,1,3
```

The default integer rectangle is:

```text
-120 <= c <= 120
1 <= m <= 120
```

The deep rectangle is:

```text
-300 <= c <= 120
1 <= m <= 260
```

## Splitting Classification Under Audit

The antisymmetric fixed-line quartic is checked to split over `Z` exactly for:

```text
c = 1, 3
c = 0  with 3 | m
c = 2  with 6 | m
c = -1 with 2 | m
c = -3 with m = 2, 3
c = -9, -11 with m = 1
```

The compact diagonal `SU(3)` slice preserves only:

```text
c = -1, 0, 1, 3
```
