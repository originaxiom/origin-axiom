## Theorem 1 (Dark Hyperbola — PROVED, all odd primes)

For every odd prime p:
  T(j,l) = 0  ⟺  j·l ≡ −4 (mod p) and (j,l) ≠ (2, p−2).
Exactly p−2 dark points. Moreover |T(2, p−2)| = √p and |T| = 1 at every
other active point — the prime-level magnitude spectrum is exactly
{0, 1, √p}.

**Proof.** p·T = Σ_{n,k} ζ^E with E = j·n(n−1)/2 − l·k(k−1)/2 + 2nk
(the 2nk cross term is the Par fold n ↦ −n).

*l ≡ 0:* Σ_k ζ^{2nk} = p·[n≡0], so p·T = p·ζ^0, T = 1 ≠ 0 (consistent:
jl ≡ 0 ≢ −4 for odd p).

*l ≢ 0* (2 invertible mod p): complete the square in k:
E = −(l/2)(k−c)² + αn² + βn + γ with
  **α = j/2 + 2/l, β = 1 − j/2, γ = l/8.**
The k-sum is a quadratic Gauss sum g(−l/2), |g| = √p ≠ 0 (classical). Then:
- α ≢ 0: the n-sum is again a Gauss sum of modulus √p ⟹ |T| = 1. ACTIVE.
- α ≡ 0, β ≢ 0: the n-sum is Σ ζ^{βn} = 0 ⟹ T = 0. DARK.
  α ≡ 0 ⟺ j·l ≡ −4; β ≢ 0 ⟺ j ≢ 2.
- α ≡ β ≡ 0: j ≡ 2, l ≡ −2; the n-sum is p·ζ^{l/8} ⟹ |T| = √p. SURVIVOR.

The hyperbola {jl ≡ −4} has p−1 points; minus the survivor: p−2 dark. ∎

  W₂ = F W₁^{∓1} F⁻¹; the |4| is intrinsic.)

## Theorem 2 (Power-Set Magnitudes — PROVED, square-free levels)

At odd square-free N = p₁⋯p_k the magnitude spectrum of T on (ℤ/N)² is
exactly {√(∏_{p∈S} p) : S ⊆ {p₁,…,p_k}} ∪ {0}.

**Proof.** CRT: n ↔ (nᵢ) splits E mod N into unit-scaled copies of the
per-prime form (uᵢ·Eᵢ, uᵢ the CRT units; n(n−1)/2 mod pᵢ depends only on
n mod pᵢ). The trace factorizes as ∏ᵢ Tᵢ. Scaling a quadratic form by a
unit preserves the classification of Theorem 1 (α, β scale by uᵢ; the dark
and survivor conditions jl ≡ −4, j ≡ 2 mod pᵢ are unchanged), so each
factor has magnitude 0, 1, or √pᵢ; products give the power set, 0 if any
factor is dark. Sharper location data for free: magnitude √(∏_S p) occurs
exactly when (j,l) ≡ (2,−2) mod every p ∈ S and is non-dark mod the rest. ∎

## Theorem 3 (Asymptotic Darkness — PROVED)
