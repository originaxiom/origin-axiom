## Result

For a determinant-one rank-2 return with trace `mu`:

```text
det(M-I) = 2 - mu
```

The torsion-one condition gives:

```text
|2-mu|=1
```

On the positive hyperbolic branch, this selects:

```text
mu=3 -> I=1/4 -> lambda/h=1
```

**`CONDITIONAL`**

mapping-torus/torsion interpretation.
