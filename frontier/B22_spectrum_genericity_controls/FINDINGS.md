# B22 -- Findings

> Logged observation, not a claim.

## Result

The `-1` sector is generic for orientation-reversing symmetric-square lifts:

```text
det(M)=-1 -> characteristic polynomial contains (t+1).
```

So the parity eigenvalue cannot by itself carry special matter or gauge content.

The quadratic `t^2-3t+1` is much more selective. It appears exactly when

```text
det(M)=-1
tr(M)=±1.
```

This is the primitive orientation-reversing/minimal-discriminant case.

## Verdict

**`STALLED`**

The generic `-1` mode kills the special parity-particle reading. The
half-step-specific object is the `A` quadratic sector, not the parity sector
alone.
