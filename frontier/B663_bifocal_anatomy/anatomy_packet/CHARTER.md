loop: read FRONTIER -> prereg+seal top cells -> agents compute -> main-seat

COMPUTE-CLOSABLE items — all remaining classified DERIVATION (escalated),

only-PARTIAL verdicts -> pause and report to the owner.

pre-seal grep E13, design-adversary); preregs sealed before compute; subagent for
