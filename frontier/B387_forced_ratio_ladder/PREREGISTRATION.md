**Committed BEFORE the level-45 sector constants are extracted or seen. Time-box: 3
sessions (campaign W2). No physics wording anywhere in this probe. Escalation rule: if a
forced ratio survives the gauntlet, STOP — promotion/voicing only at a dedicated
adversarial session with the owner present.**

## The object

3-block = +(1/(6φ))√−3, i.e. (0,0,−1/12,+1/12). Level-45: the analogous pair-graded
sector constants of the (1,2) pair at level 45, extracted from the banked sweep45 pair
tables (fp identification; machinery `fp_engine.identify` / `sweep45.run_level`) — the
graded combinations at the level-45 slot exponents (identified by the same mult-1

## The registered bet (2b): const(45) / const(15) ∈ one of

  (R1) 1 — identity transport (the prior: the m=1 single-seed 1/4 transports identically,
       PR #484);
  (R2) a unit of ℤ[φ]: φ, 1/φ, φ², 1/φ², −1 times these (the golden-unit family);
  (R3) a ratio of Pisano data: π(45)/π(15) = 120/40 = 3, or its inverse, or ±3^{±1}·(R2);
  (R4) the level ratio family: 3, 1/3, 9, 1/9 times (R1–R2).

KILL: the ratio is none of these (bank the exact value — an unregistered exact ratio is
still a banked fact, but it is NOT called "forced"; the ladder then needs level 135 before
any claim of pattern). VACUITY CHECK: the candidate set above contains ≥ 24 distinct
values, so a match is a real selection, not a tautology; and the null "no sector at 45"
is already excluded by the banked existence data (level-45 rungs banked).

## The gauntlet (2c; runs ONLY if a ratio from the registered set survives)

Pre-registered nulls: (i) permuted-grading null — the same assembly with the grading signs
permuted must NOT give a registered-set ratio; (ii) wrong-exponent control — slot exponents
shifted by +1 must not; (iii) wrong-pair control — the (1,3)-pair (dark at 15) analog must
not produce the same ratio. Any null failure kills the "forced" reading (bank as
coincidence-of-assembly).

## Outcomes

the gauntlet verdict, then STOP per the escalation rule.
