## Result

The projective return matching rule gives a Lucas hierarchy:

```text
(lambda/h)^2 = L_n - 2,  n even
1, 5, 16, 45, 121, ...
```

If the primitive original sector is required, the first hyperbolic member is
`n=2`, giving:

```text
lambda/h=1
```

## Verdict

**`CONDITIONAL`**

The renormalization route selects `lambda/h=1` only if the primitive original
sector is treated as the ground selector and higher Lucas values are treated as
excited sectors.
