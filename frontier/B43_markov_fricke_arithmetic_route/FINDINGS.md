## Result

The tangent quadratic discriminant is:

```text
Delta = mu^2 - 4 = 16I(I+1)
```

If `mu` is required to be an integer hyperbolic trace, the smallest positive
nonsquare discriminant is:

```text
mu=3 -> Delta=5 -> I=1/4
```

Without integer trace, `Delta` is continuous and no arithmetic minimum is
available.

## Verdict

**`CONDITIONAL`**

The discriminant route recovers S1 only after the tangent integer-trace filter
is assumed.
