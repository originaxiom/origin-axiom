This archive contains every completed artifact from the governed frontier campaign:

- Cell 1 — Universal Z/11 charge transport
- Cell 1B — Localized charge carriers
- Cell 1C — Defect kinematics
- Cell 2 — Observer-flow closure and protocol dependence
- Cell 3 — Fixed-character local atlas and completeness gate
- Cell 4 — Degree-4 spectral robustness and measurement audit

## Important status boundary

Cell 5 (Ghost Scanner Completion) was started but had not yet produced a completed,
packaged result at the time this archive was assembled. It is therefore not included
as a completed cell. The existing repository B547 ghost-scanner work is separate from
this governed campaign package.

## Structure

- `cells/` — extracted, non-duplicated artifacts grouped by cell
- `original_cell_packages/` — original ZIP packages preserved byte-for-byte
- `MASTER_MANIFEST.json` — SHA-256 checksum and size for every included file
- `CAMPAIGN_STATUS.md` — concise result/status ledger

No repository files were modified while assembling this archive.
