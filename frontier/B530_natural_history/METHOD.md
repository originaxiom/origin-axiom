# The Listening Method — proper listening without anticipation

**Registered 2026-07-12 (owner-approved posture).** The method for B530's natural history. It is a *map, not
an agenda*: it fixes how to look, never what to find. Firewalled throughout; nothing physics-shaped.

## The rules
1. **No target, no prereg outcome, no hypothesis, no verdict.** We ask the object nothing; we only look.
2. **Neutral, complete invariant-census.** Cover the object's standard invariants in a fixed order the seat
   does *not* get to curate. (Curating the list toward the "interesting" ones is the subtlest anticipation.)
3. **Report everything — flat and rich alike.** A flat invariant ("balance = 3, unremarkable") is data;
   skipping it because it's expected to be boring is anticipation. Bank the flat ones too.
4. **Depth is led by the object.** When an invariant opens surprising structure, follow it as deep as it goes;
   when flat, note and move on. The seat never forces depth where it hopes for it.
5. **Both currents.** Upstream (to the roots / fields) and downstream (to the dynamics), but connections are
   drawn only when the object's structure genuinely bridges — never manufactured.
6. **The discipline holds.** Compute the discriminating fact (never assert/cite/proxy — [[compute-the-discriminating-fact]]);
   firewall (no physics reading — [[listen-to-the-object-not-physics]]); cross-seat verify; and the
   *anticipation-check* before every bank: "did the object hand me this, or did I steer to it?" — flag if steered.
7. **No closure.** The census ends when the invariants are covered, not when we've found what we wanted.
