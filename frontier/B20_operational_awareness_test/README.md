# Probe B20 -- Operational awareness test

> **Speculative frontier work.** Logged observations only. This probe does not
> make consciousness or qualia claims.

## Question

Can the trace map satisfy a strict operational definition of awareness-like
feedback?

## Operational Definitions

```text
memory = an invariant preserved by dynamics
feedback = state-dependent update
self-model = update rule explicitly reads its invariant/classifier
recursive sector = lifted dynamics contains the original A-sector
```

## Result

The trace map has invariant-memory and recursive sector structure. It does not
explicitly read `I`; its formula is the same on every `I`-surface. Orbit behavior
can depend on the surface through initial conditions, but that is classification,
not read-and-respond self-modeling.

## Verdict

`STALLED`: operational invariant-memory yes; operational self-model no.
