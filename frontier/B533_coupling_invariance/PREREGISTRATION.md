# B533 — Coupling Invariance

## Question

The object σ can be observed from different vantage points (factors u of
the fixed-point word). Each vantage point induces a return-word substitution
A_u with its own structure. Which dimensionless quantities are independent
of the observation point (intrinsic to the object), and which depend on
the coupling (the choice of u)?

This is Gate 1 of 3 toward the question "can the object predict SM values
given scale as coupling?"

## Gates

- Gate 1 (this probe): coupling-invariance — which ratios are intrinsic?
- Gate 2 (future): dimensional bridge — what form does f(σ, scale) → value take?
- Gate 3 (future, only if Gate 1 passes): discriminating match/mismatch with SM

## Predictions (pre-registered)

1. **Letter-frequency ratios are universal** (trivially, by partitioning).
   f_a/f_b = φ·√φ, f_A/f_B = φ, f_a/f_A = 1/√φ — same for all u.

2. **Return-word-frequency ratios may vary** — the Perron eigenvector of A_u
   depends on the structure of the induced system, not just on M.

3. **The closing power q (1 vs 2) varies** with u — this is established by B530.

4. **Self-recovery**: some factors u may give canonical codes equivalent to σ
   (the induced system IS σ up to relabeling); if all do, the object is
   fully self-similar; if only some do, the coupling point matters.

## Null hypothesis

The observation point is a gauge choice — all dimensionless ratios are
universal. If this fails, the coupling carries information, and the
question "which coupling?" becomes meaningful.

## Method

1. Census of observation points (factors u, length 1–4)
2. Perron eigenvector + Parikh matrix → effective letter frequencies
3. Return-word-frequency gap labels (observation-dependent spectral data)
4. Self-recovery test (canonical codes vs σ)
5. Transition probabilities between return words

## Deliverable

A table: for each dimensionless quantity, UNIVERSAL or VARIES (with range).
