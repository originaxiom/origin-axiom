# Mechanism classes

The 20 mathematizable emergence-paths (E1–E20) and 5 philosophical paths (P1–P5)
are grouped into 12 mechanism classes (A–L). Classes are not mutually exclusive —
some paths sit at the boundary of two — but the grouping helps avoid attacking
five variants of the same mechanism in a row.

The class structure exists for **two purposes**:

1. *Coverage.* The first-batch selection picks across classes; later batches should
   keep widening, not deepen one class until the others are still untouched.
2. *Diagnosis.* If the same kind of wall appears in every probe within a class,
   the wall is about the class. If it appears across classes, the wall is about
   the question.

## The classes

### Class A — Fixed-point / attractor dynamics
**Idea:** "nothing" is an unstable fixed point of some dynamics; the stable
attractor *is* reality. Selection by dynamics rather than aesthetics.
**Paths:** E1 (self-reference iteration → φ), E2 (continued-fraction / Gauss map).
**Status in this project:** partially explored — `../frontier/B1`, `B4`.

### Class B — Quantum zero-point mechanisms
**Idea:** quantum-mechanical zero-point energy or vacuum fluctuations are bounded
below by some non-zero quantity; vacuum cannot be empty.
**Paths:** E3 (vacuum-mode cancellation failure — the original framing), E4
(Heisenberg uncertainty as floor), E5 (Vilenkin / Hartle–Hawking tunneling).
**Status:** E3 narrow forms are `DEAD` (D1–D7); E4, E5 untouched.

### Class C — Topological / geometric obstruction
**Idea:** non-trivial topology forbids the void; minimum topological content is
forced.
**Paths:** E6 (fundamental group / homology), E7 (causal sets), E8 (univalent
foundations / higher homotopy).
**Status:** E6 partially explored via the figure-eight characterization (P1–P13).

### Class D — Symmetry / variational
**Idea:** the symmetric or trivial vacuum is unstable to perturbation or fails to
be an extremum of some action.
**Paths:** E9 (spontaneous symmetry breaking), E10 (action principle).

### Class E — Statistical / informational
**Idea:** statistical or information-theoretic counting forces emergence
("nothing" has measure zero among configurations).
**Paths:** E11 (entropic emergence), E12 (it-from-bit), E13 (measure-theoretic).

### Class F — Categorical / formal
**Idea:** "nothing" as a mathematical object (initial object, empty type) has its
own structure that generates more.
**Paths:** E14 (initial-object / empty-type).

### Class G — Boundary / holographic
**Idea:** bulk emerges from boundary; "outside of nothing" carries structure.
**Paths:** E15 (bulk-from-boundary).

### Class H — Renormalization / scale-dependent
**Idea:** the answer depends on scale — empty at one scale, structured at another.
**Paths:** E16 (RG flow), E17 (extended objects).

### Class I — Bootstrap / consistency
**Idea:** only some "somethings" are self-consistent; nothing isn't; consistency
is the selector.
**Paths:** E18 (self-consistency / bootstrap).

### Class J — Number-theoretic
**Idea:** certain equations admit no zero solution; the smallest non-zero solution
is reality.
**Paths:** E19 (Diophantine non-existence).

### Class K — Cosmological
**Idea:** empty space is not a stable cosmological state.
**Paths:** E20 (de Sitter / positive Λ).

### Class L — Philosophical (separate register)
**Idea:** the argument is non-mathematizable (or not yet mathematized). Lives in
`philosophical/`, kept clearly separate from claims-bearing work.
**Paths:** P1 (negation-as-creation), P2 (Gödelian), P3 (anthropic), P4 (process),
P5 (apophatic).

## What's *not* a class

We deliberately do not have a class for "the universe is X" (where X is some
specific structure like the figure-eight knot, a particular Lie algebra, a
specific TQFT). Those are *answers*, not mechanisms. The mechanism is the
*reason* a specific structure would be selected. The whole project's recurring
failure mode was naming the answer without supplying the mechanism — this class
structure is built to keep the focus on mechanism.
